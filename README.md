# Ayyappa Pradakshina Tracker

A simple, easy-to-use web application designed to help devotees track their progress while performing the Ayyappa Pradakshina by chanting the 108 names of Lord Ayyappa (Sharanu Gosha).

## Features

*   **108-Step Checklist:** Easily track each of the 108 chants with a simple checkbox.
*   **Progress Visualization:** A dynamic progress bar shows your completion percentage as you go.
*   **Audio Assistance:** Each name has an associated audio file that can be played to help with pronunciation and recitation.
*   **Bilingual Support:** Switch between **English** and **Telugu** versions of the Sharanu Gosha with a convenient toggle.
*   **Session Timing:** The application automatically records the start and end times of your pradakshina.
*   **Completion History:** Your completed sessions are automatically saved, allowing you to track the date and duration of each pradakshina.
*   **Dark/Light Mode:** Choose between a light or dark theme for comfortable viewing in any lighting condition.
*   **Persistent Settings:** Your preferred theme, language, and completion history are saved in your browser's local storage for a seamless experience every time you visit.
