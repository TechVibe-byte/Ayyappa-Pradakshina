# Ayyappa Pradakshina

A simple web application to track the 108 steps of Ayyappa Pradakshina, with gamification features to keep you motivated.

## Features

*   **108 Steps Checklist:** Track your progress through the Sarana Ghosha.
*   **Progress Bar:** Visualize your completion percentage.
*   **Time Tracking:** See your start and end times for each session.
*   **Multi-Language:** Switch between English and Telugu.
*   **Dark/Light Mode:** Choose your preferred theme.
*   **Completion History:** Keep a record of all your completed pradakshinas.
*   **Streaks:** Track your consecutive daily completions to stay motivated.
*   **Badges:** Unlock achievement badges for reaching milestones (e.g., 1, 5, 10 completions).
*   **Leaderboard:** A personal leaderboard to see your total number of completions.

## How to Use

1.  Open the `index.html` file in your web browser.
2.  Check the boxes as you complete each step.
3.  Your progress, time, and streak will be automatically tracked.
4.  View your accomplishments in the History, Badges, and Leaderboard sections.
5.  Use the "Clear History" button to reset all your data.

## Deployment

This is a static website built with HTML, CSS, and JavaScript. You can easily deploy it using [GitHub Pages](https://pages.github.com/):

1.  Create a new public repository on GitHub.
2.  Push the project files (`index.html`, `styles.css`, `script.js`, and the `images` folder) to the repository.
3.  In the repository settings, go to the "Pages" section.
4.  Select the `main` branch as the source and click "Save".
5.  Your website will be live at `https://<your-username>.github.io/<your-repo-name>/`.
