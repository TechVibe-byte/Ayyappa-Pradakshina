# Ayyappa Pradakshina

A simple web application to track the 108 steps of Ayyappa Pradakshina, with gamification features to keep you motivated.

## Features

*   **108 Steps Checklist:** Track your progress through the Sarana Ghosha.
*   **Progress Bar:** Visualize your completion percentage.
*   **Time Tracking:** See your start and end times for each session.
*   **Multi-Language:** Switch between English and Telugu.
*   **Dark/Light Mode:** Choose your preferred theme.
*   **Completion History:** Keep a record of all your completed pradakshinas.
*   **Streaks:** Track your consecutive daily completions to stay motivated.
*   **Badges:** Unlock achievement badges for reaching milestones (e.g., 1, 5, 10 completions).
*   **Leaderboard:** A personal leaderboard to see your total number of completions.


## Technologies Used

*   HTML
*   CSS
*   JavaScript
